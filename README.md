# Trackip-X
![Trackip-X preview](Trackip-X.jpg)

## introduction
This is a tool to track someone's IP address very accurately

## Instalations
```
$ pkg update && pkg upgrade
$ pkg install python
$ pkg install git
$ git clone https://github.com/Whomrx666/Trackip-X.git
$ cd Trackip-X 
$ pip install -r requirements.txt
$ python -m pip install geocoder
$ python Trackip-X.py
```

## Instructions
- **First**: Install tools according to the instructions above
- **Second**: Then enter the IP address that you have obtained into the tool
- **Done**: And you will see various kinds of information about the IP
